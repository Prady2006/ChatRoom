Its is a basic chat application.
File client2.py needs to be run on the client side .
File server-1-to-1.py lets clients to chat to common server but clients can't chat among themselves .Server will present a list of connected users and chat with them in a one on one basis .
File server_broadcast_1-to-1.py is a group chat , with a functionality to send DM too . Just write the username and then the message it would be a DM then .
